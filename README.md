# Campusify

Dasbor akademik lengkap untuk mahasiswa Indonesia. Kelola nilai, jadwal, tugas, keuangan, dan target akademik dalam satu aplikasi.

## ✨ Fitur

- 📊 **Manajemen Nilai** - Lacak nilai dan hitung IP/IPK otomatis
- 📅 **Jadwal Kuliah** - Atur dan lihat jadwal perkuliahan  
- ✅ **Tugas Akademik** - Kelola tugas dengan deadline dan prioritas
- 💰 **Keuangan** - Catat pemasukan dan pengeluaran
- 👤 **Kehadiran** - Pantau kehadiran per mata kuliah
- 🎯 **Target Akademik** - Tetapkan dan pantau target pencapaian
- 🌙 **Mode Gelap** - Dukungan tema terang dan gelap
- 📱 **Responsive** - Optimal di desktop dan mobile
- 📤 **Export/Import** - Backup data via Excel atau JSON

## 🚀 Demo

Aplikasi ini dapat diakses di: [https://campusify.vercel.app](https://campusify.vercel.app)

## 💻 Development

```bash
# Clone repository
git clone <repository-url>
cd campusify

# Install dependencies (optional, untuk development)
npm install

# Run local server
npm run dev
# atau
npx serve . -l 3000

# Akses di browser
open http://localhost:3000
```

## 📦 Deploy ke Vercel

1. Push code ke GitHub repository
2. Connect repository ke Vercel
3. Deploy otomatis akan berjalan
4. Aplikasi siap digunakan!

## 🏗️ Teknologi

- **Frontend**: Vanilla JavaScript (ES6+), CSS3, HTML5
- **Storage**: LocalStorage browser
- **Hosting**: Vercel static hosting
- **Charts**: Custom canvas-based charts
